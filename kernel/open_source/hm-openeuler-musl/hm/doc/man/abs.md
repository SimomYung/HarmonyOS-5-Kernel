# abs()
Return the absolute value of an integer

## Synopsis
```c
#include <stdlib.h>

int abs(int a);
```

## Arguments
- **a**
An integer number you want to get absolute value of.

## Library
libc

## Description
The abs() function returns the absolute value of integer argument *a*.

## Returns
The absolute value of input argument.
