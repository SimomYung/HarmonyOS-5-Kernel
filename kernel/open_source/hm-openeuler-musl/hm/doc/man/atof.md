# atof()
Convert a string into a double

## Synopsis
```c
#include <stdlib.h>

double atof(const char *s)
```

## Arguments
- **s**
A pointer to the string to be converted.

## Library
libc

## Description
atof() is used to convert a string to a double.

## Returns
The atof() returns double converted from string argument.
