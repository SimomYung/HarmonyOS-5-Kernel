# atoi()
Convert a string to an integer

## Synopsis
```c
#include <stdlib.h>

int atoi(const char *s)
```

## Arguments
- **s**
A pointer to the string to be converted.

## Library
libc

## Description
atoi() is used to convert a string to an integer.

## Returns
Return an integer converted from string argument.
