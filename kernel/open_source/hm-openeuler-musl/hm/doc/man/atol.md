# atol()
Convert a string into a long integer

## Synopsis
```c
#include <stdlib.h>

long atol(const char *s);
```

## Arguments
- **s**
A pointer to the string to be converted.

## Library
libc

## Description
atol() is used to convert a string into a long integer.

## Returns
It returns the converted value on success.
