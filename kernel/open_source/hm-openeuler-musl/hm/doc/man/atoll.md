# atoll()
Convert a string into a long long integer

## Synopsis
```c
#include <stdlib.h>

long long atoll(const char *s);
```

## Arguments
- **s**
A pointer to the string to be converted.

## Library
libc

## Description
The atoll() is used to convert a string into a long long integer.

## Returns
Return a long long integer converted from string argument.
