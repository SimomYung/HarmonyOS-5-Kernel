# cacosl()

Complex arc cosine

## Synopsis
```c
#include <complex.h>

long double complex cacosl(long double complex z);
```

## Arguments
- **z**
Independent variable *z*.

## Library

libc

## Description

This function calculates the complex arc cosine of *z*. The real part of the result is chosen in the interval [0, pi].

## Returns

The arc cosine value of the given complex number.
