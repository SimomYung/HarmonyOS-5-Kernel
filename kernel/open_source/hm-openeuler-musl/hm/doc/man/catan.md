# catan()

Complex arc tangent

## Synopsis
```c
#include <complex.h>

double complex catan(double complex z);
```

## Arguments
- **z**
Independent variable *z*

## Library

libc

## Description

This function calculates the complex arc tangents of *z*. The real part of the result is chosen in the interval [-pi/2, pi/2].

## Returns

The arc tangent value of the given complex number.
