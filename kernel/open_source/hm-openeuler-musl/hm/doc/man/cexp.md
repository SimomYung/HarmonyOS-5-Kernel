#  cexp(), cexpf(), cexpl()
Compute the complex base\-e exponential of the given complex number

## Synopsis
```c
#include <complex.h>

double complex cexp(double complex c);
float complex cexpf(float complex c);
long double complex cexpl(long double complex c);
```

## Arguments
- **c**
A complex number.

## Library
libc

## Description
These functions compute the complex base\-e exponential of the given complex number *c*.

## Returns
The complex base\-e exponential of the given complex number.
