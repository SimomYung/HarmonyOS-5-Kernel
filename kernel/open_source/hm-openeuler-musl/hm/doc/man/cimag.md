# cimag(), cimagf(), cimagl()
Return the imaginary part of the given complex number

## Synopsis
```c
#include <complex.h>

double cimag(double complex c);
float cimagf(float complex c);
long double cimagl(long double complex c);
```

## Arguments
- **c**
A complex number.

## Library
libc

## Description
These functions return the imaginary part of the given complex number *c*.

## Returns
The imaginary part of the given complex number.
