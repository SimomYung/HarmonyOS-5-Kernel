# creal(), crealf(), creall()
Return the real part of the given complex number

## Synopsis
```c
#include <complex.h>

double creal(double complex c);
float crealf(float complex c);
long double creall(long double complex c);
```

## Arguments
- **c**
A complex number.

## Library
libc

## Description
These functions return the real part of the given complex number *c*.

## Returns
The real part of the given complex number.
