# csin(), csinf(), csinl()
Compute the sine value of the given complex number

## Synopsis
```c
#include <complex.h>

double complex csin(double complex c);
float complex csinf(float complex c);
long double complex csinl(long double complex c);
```

## Arguments
- **c**
A complex number.

## Library
libc

## Description
These functions compute the sine value of the given complex number *c*.

## Returns
The sine value of the given complex number.
