# csinh(), csinhf(), csinhl()
Compute the hyperbolic sine value of the given complex number

## Synopsis
```c
#include <complex.h>

double complex csinh(double complex c);
float complex csinhf(float complex c);
long double complex csinhl(long double complex c);
```

## Arguments
- **c**
A complex number.

## Library
libc

## Description
These functions compute the hyperbolic sine value of the given complex number *c*.

## Returns
The hyperbolic sine value of the given complex number.
