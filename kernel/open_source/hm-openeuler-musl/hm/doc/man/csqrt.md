# csqrt(), csqrtf(), csqrtl()
Compute the square root of the given complex number

## Synopsis
```c
#include <complex.h>

double complex csqrt(double complex c);
float complex csqrtf(float complex c);
long double complex csqrtl(long double complex c);
```

## Arguments
- **c**
A complex number.

## Library
libc

## Description
These functions compute the square root of the given complex number *c*.

## Returns
The square root of the given complex number.
