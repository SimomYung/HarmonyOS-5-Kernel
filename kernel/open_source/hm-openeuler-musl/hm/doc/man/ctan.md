# ctan(), ctanf(), ctanl()
Compute the tangent value of the given complex number

## Synopsis
```c
#include <complex.h>

double complex ctan(double complex c);
float complex ctanf(float complex c);
long double complex ctanl(long double complex c);
```

## Arguments
- **c**
A complex number.

## Library
libc

## Description
These functions compute the tangent value of the given complex number *c*.

## Returns
The tangent value of the given complex number.
