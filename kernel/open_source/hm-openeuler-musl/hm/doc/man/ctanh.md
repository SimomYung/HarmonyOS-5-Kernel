#  ctanh(), ctanhf(), ctanhl()
Compute the hyperbolic tangent value of the given complex number

## Synopsis
```c
#include <complex.h>

double complex ctanh(double complex c);
float complex ctanhf(float complex c);
long double complex ctanhl(long double complex c);
```

## Arguments
- **c**
A complex number.

## Library
libc

## Description
These functions compute the hyperbolic tangent value of the given complex number *c*.

## Returns
The hyperbolic tangent value of the given complex number.
