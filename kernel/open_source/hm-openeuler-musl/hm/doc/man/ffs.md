# ffs()
Obtain the first bit 1

## Synopsis
```c
#include <atomic.h>
#include <strings.h>

int ffs(int i);
```

## Arguments

- **i**
The input parameter.


## Library
libc

## Description
Function ffs() obtains the first bit (1) in the number *i*.

## Returns

- **Integer**
Where the first bit 1 is.
