# getpid()
Get the process ID

## Synopsis
```c
#include <unistd.h>

pid_t getpid(void);
```

## Arguments
N/A

## Library
libc

## Description

The getpid() function returns the process ID of the calling process.

## Returns

- **pid**
The process ID queried. The getpid() function is always successful.
