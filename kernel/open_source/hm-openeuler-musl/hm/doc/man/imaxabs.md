# imaxabs()
Return the absolute value of an appropriate integer value

## Synopsis
```c
#include <inttypes.h>

intmax_t imaxabs(intmax_t a);
```

## Arguments
- **a**
An appropriate integer number you want to get absolute value of.

## Library
libc

## Description
The imaxabs() function returns the absolute value of appropriate integer argument *a*.

## Returns
The absolute value of input argument.
