# isalnum()
Check to see if the character is an alphanumeric character

## Synopsis
```c
#include <ctype.h>

int isalnum(int c);
```

## Arguments 

- **c**
  The character to check.


## Library
libc

## Description
The isalnum() function checks to see if the character is an alphanumeric character.

## Returns

- **non-zero**
  This character is an alphanumeric character.
&nbsp;
- **0**
  This character is not an alphanumeric character.
