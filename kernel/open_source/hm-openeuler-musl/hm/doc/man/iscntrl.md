# iscntrl()

Check to see if the character is a control character

## Synopsis

```c
#include <ctype.h>

int iscntrl(int c);
```

## Arguments

- **c**
  The character to check.

## Library

libc

## Description

The iscntrl() function checks to see if the character is a control character.

## Returns

- **non-zero**
  This character is a control character.
&nbsp;
- **0**
  This character is not a control character.
