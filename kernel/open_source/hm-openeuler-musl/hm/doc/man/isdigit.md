# isdigit()

Check to see if the character is a decimal digit

## Synopsis

```c
#include <ctype.h>

int isdigit(int c);
```

## Arguments

- **c**
  The character to check.

## Library

libc

## Description

The isdigit() function checks to see if the character is a decimal digit.

## Returns

- **non-zero**
  This character is a decimal digit.
&nbsp;
- **0**
  This character is not a decimal digit.
