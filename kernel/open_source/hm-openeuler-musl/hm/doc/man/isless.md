# isless()

Check to see if the first argument is less than the second argument.

## Synopsis

```c
#include <ctype.h>

bool isless(float x, float y);
```

## Arguments

- **x**
  The first floating point number to compare.
  &nbsp;
- **y**
  The second floating point number to compare

## Library

libc

## Description

The isless() function checks to see if the first argument is less than the second argument.

## Returns

- **true**
  This first argument is less than the second argument.
&nbsp;
- **false**
  the first argument is not less than the second argument.
