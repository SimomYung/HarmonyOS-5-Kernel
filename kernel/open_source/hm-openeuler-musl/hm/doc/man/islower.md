# islower()

Check to see if the character is a lowercase character

## Synopsis

```c
#include <ctype.h>

int islower(int c);
```

## Arguments

- **c**
  The character to check.

## Library

libc

## Description

The islower() function checks to see if the character is a lowercase character.

## Returns

- **non-zero**
  This character is a lowercase character.
&nbsp;
- **0**
  This character is not a lowercase character.
