# isprint()

Check to see if the character is printable character including space

## Synopsis

```c
#include <ctype.h>

int isprint(int c);
```

## Arguments

- **c**
  The character to check.

## Library

libc

## Description

The isprint() function checks to see if the character is printable character including space.

## Returns

- **non-zero**
  This character is a printable character including space.
&nbsp;
- **0**
  This character is not a printable character including space.
