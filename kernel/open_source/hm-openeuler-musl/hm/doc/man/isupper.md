# isupper()

Check to see if the character is a uppercase character

## Synopsis

```c
#include <ctype.h>

int isupper(int c);
```

## Arguments

- **c**
  The character to check.

## Library

libc

## Description

The isupper() function checks to see if the character is a uppercase character.

## Returns

- **non-zero**
  This character is a uppercase character.
&nbsp;
- **0**
  This character is not a uppercase character.
