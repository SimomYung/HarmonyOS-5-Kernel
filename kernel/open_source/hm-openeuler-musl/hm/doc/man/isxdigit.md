# isxdigit()

Check to see if the character is a hexadecimal digit character

## Synopsis

```c
#include <ctype.h>

int isxdigit(int c);
```

## Arguments

- **c**
  The character to check.

## Library

libc

## Description

The isxdigit() function checks to see if the character is a hexadecimal digit character.

## Returns

- **non-zero**
  The character is a hexadecimal digit character.
&nbsp;
- **0**
  This character is not a hexadecimal digit character.
