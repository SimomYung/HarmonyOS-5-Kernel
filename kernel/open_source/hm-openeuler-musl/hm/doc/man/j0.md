# j0(), j0f()
Bessel functions

## Synopsis
```c
#include <math.h>

double j0(double x);
float j0f(float x);

```

## Arguments
- **x**
The number to be calculated in Bessel functions.
&nbsp;

## Library
libc

## Description
The j0 and j0f functions return Bessel functions of *x* of the first kind of order 0.

## Returns
- **Bessel value of the first kind of order 0 for x**
If success.
&nbsp;

- **0**
If result underflows or *x* is too large.
&nbsp;

- **NaN**
If *x* is a NaN.
