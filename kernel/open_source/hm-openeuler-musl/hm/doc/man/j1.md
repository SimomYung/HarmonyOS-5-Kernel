# j1(), j1f()
Bessel functions

## Synopsis
```c
#include <math.h>

double j1(double x);
float j1f(float x);

```

## Arguments
- **x**
The number to be calculated in Bessel functions.
&nbsp;

## Library
libc

## Description
The j1 and j1f functions return Bessel functions of *x* of the first kind of order 1.

## Returns
- **Bessel value of the first kind of order 1 for x**
If success.
&nbsp;

- **0**
If result underflows or *x* is too large.
&nbsp;

- **NaN**
If *x* is a NaN.
