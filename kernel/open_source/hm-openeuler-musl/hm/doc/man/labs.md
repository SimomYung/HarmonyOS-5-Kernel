# labs()
Return the absolute value of a long integer

## Synopsis
```c
#include <stdlib.h>

long int labs(long int a);
```

## Arguments
- **a**
A long integer number you want the absolute value of.

## Library
libc

## Description
The labs() function returns the absolute value of long integer argument *a*.

## Returns
The absolute value of input argument.
