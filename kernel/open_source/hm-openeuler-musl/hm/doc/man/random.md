# random(), random\_r()
Generate a pseudo-random long integer

## Synopsis
```c
#include <stdlib.h>

long random(void);
int random_r(struct random_data *data, long *result);
```

## Arguments

- **data**
The pointer contains the state information for generating the pseudo-random num.
&nbsp;
- **result**
This argument stores the result(i.e., the pseudo-random number) which generated by the function.


## Library
libc

## Description
The random() function generates a pseudo-random long integer which range between 0 and RAND\_MAX. The period of results between calls is about (2^31) - 1.

The random\_r() function is the reentrant version of the random() function, it generates a pseudo-random long integer as well. The argument *data* is a pointer which stores the state information between calls. If called with the same value in the argument, the random\_r() function will generate the same pseudo-random num.

## Returns
The random() function shall return a pseudo-random integer range between 0 and RAND\_MAX.

The random\_r() function shall return as follow:

- **0**
The pseudo-random num was generated successfully.
&nbsp;
- **-1**
An error occurred.

## Errors
N/A

## Examples
```c
#include <stdlib.h>
#include <stdio.h>

int main(int argc, char *argv[])
{
        int i;
        unsigned int seed = 0x1234;

        srandom(seed);
        for (i = 0; i < 10; ++i) {
                int randomNum = random();
                printf("The %d num is %d from random\n", i, randomNum);
        }

        return 0;
}
```
