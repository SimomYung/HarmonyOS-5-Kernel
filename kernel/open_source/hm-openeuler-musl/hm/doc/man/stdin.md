# stdin, stdout, stderr
stdin, stdout, stderr - standard I/O stream

## Synopsis
```c
#include <stdio.h>

extern FILE *stdin;
extern FILE *stdout;
extern FILE *stderr;
```

## Library
libc

## Description
The stdin refers to standard input; the stdout refers to standard output; and the stderr refers to standard error.
