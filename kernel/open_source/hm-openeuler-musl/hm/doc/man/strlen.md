# strlen()
Return the length of a string

## Synopsis
```c
#include <strings.h>

size_t strlen(const char *s);
```

## Arguments

- **s**
A pointer to the string.

## Library
libc

## Description
Return the length of the string *s*, not including the terminating null byte ('\0').

## Returns
The number of characters in *s*.
