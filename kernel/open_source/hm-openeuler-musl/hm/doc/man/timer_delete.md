# timer_delete()
Delete a POSIX per-process timer

## Synopsis
```c
#include <time.h>;

int timer_delete(timer_t timerid);
```

## Arguments
- **timerid**
The ID of the timer.

## Library
libc

## Description
The timer_delete() function deletes the timer whose ID is given in *timerid*. If the timer was armed at the time of the call, it is disarmed before being deleted. The treatment of any pending signal generated by the deleted timer is unspecified.

## Returns
- **0**
Success.
&nbsp;
- **-1**
An error occurred.

## Errors
-  **EINVAL**
*timerid* is not a valid timer ID.
