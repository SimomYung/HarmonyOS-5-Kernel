# div()
Calculate quotient and remainder of an integer division

## Synopsis
```c
#include <stdlib.h>

div_t div (int numer, int denom)
```

## Arguments
- **numer**
The numerator in the division.
&nbsp;
- **denom**
The denominator.

## Library
libc

## Description
The div() function calculates quotient and remainder of an integer division.

## Returns
A div_t structure containing the quotient and remainder.

N/A.
