# isalpha()

Check to see if the character is an alphabetic character

## Synopsis
```c
#include <ctype.h>

int isalpha(int c);
```

## Arguments 

- **c**
  The character to check.


## Library
libc

## Description
The isalpha() function checks to see if the character is an alphabetic character.

## Returns

- **non-zero**
  This character is an alphabetic character.
&nbsp;
- **0**
  This character is not an alphabetic character.
