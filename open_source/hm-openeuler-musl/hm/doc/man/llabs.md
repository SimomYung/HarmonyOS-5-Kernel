# llabs()
Return the absolute value of a long long integer

## Synopsis
```c
#include <stdlib.h>

long long int llabs(long long int a);
```

## Arguments
- **a**
A long long integer number you want to get absolute value of.

## Library
libc

## Description
The llabs() function returns the absolute value of long long integer argument *a*.

## Returns
The absolute value of input argument.
